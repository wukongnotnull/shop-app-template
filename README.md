# shop-fe-vue2

## 项目介绍
该项目是基于 Vue2.x  + ElementUI2.x 的电商前端项目，该项目包含游客首页（前台）和管理员首页（后台）。
该项目实现了一般电商的基础功能，方便大家以该项目作为模板，进行二次开发。
